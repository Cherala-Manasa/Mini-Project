HTML Beginner Quiz

Test your beginner knowledge of HTML.

1. What is the correct HTML element for inserting a line break?

A. <br>
B. <break>
C. <lb>
D. <linebreak>
Incorrect. Use <br> for inserting a line break.
Correct! Use <br> for inserting a line break.

2. Which tag is used to define an image in HTML?

A. <img>
B. <picture>
C. <photo>
D. <image>
Incorrect. Use <img> to define an image in HTML.
Correct! Use <img> to define an image in HTML.

3. How do you add a comment in HTML?

A. <!-- Comment -->
B. <comment> Comment </comment>
C. <# Comment #>
D. </* Comment */>
Incorrect. Use <!-- Comment --> to add a comment in HTML.
Correct! Use <!-- Comment --> to add a comment in HTML.

4. What does the <meta> tag define in HTML?

A. Metadata about the document
B. A script for the document
C. A style sheet link
D. A hyperlink
Incorrect. The <meta> tag defines metadata about the document.
Correct! The <meta> tag defines metadata about the document.

5. How do you make a list item in HTML?

A. <li>
B. <list>
C. <item>
D. <ul>
Incorrect. Use <li> to make a list item.
Correct! Use <li> to make a list item.

6. What element is used to define a table in HTML?

A. <table>
B. <tab>
C. <tr>
D. <tbody>
Incorrect. Use <table> to define a table.
Correct! Use <table> to define a table.

7. Which tag is used to define a hyperlink in HTML?

A. <a>
B. <link>
C. <href>
D. <url>
Incorrect. Use <a> to define a hyperlink.
Correct! Use <a> to define a hyperlink.

8. How do you include an external CSS file in HTML?

A. <style src="file.css">
B. <css href="file.css">
C. <link rel="stylesheet" href="file.css">
D. <script src="file.css">
Incorrect. Use <link rel="stylesheet" href="file.css"> to include an external CSS file.
Correct! Use <link rel="stylesheet" href="file.css"> to include an external CSS file.

9. What is the purpose of the <head> element in HTML?

A. Contains metadata and links to resources
B. Contains the main content of the page
C. Defines the document’s title
D. Contains footer information
Incorrect. The <head> element contains metadata and links to resources.
Correct! The <head> element contains metadata and links to resources.

10. Which tag is used to define a table header in HTML?

A. <th>
B. <header>
C. <td>
D. <tr>
Incorrect. Use <th> to define a table header.
Correct! Use <th> to define a table header.
