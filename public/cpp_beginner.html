C++ Language Beginner Quiz

Test your beginner knowledge of C++ programming Language.

1. Which of the following is the correct syntax to include a header file in C++?

A. #include <header.h">
B. #include "header.h"
C. include <header.h">
D. include "header.h"
Incorrect. The correct syntax is #include <header.h> or #include "header.h".
Correct! The correct syntax is #include <header.h> or #include "header.h".

2. What is the default access specifier for class members in C++?

A. Public
B. Protected
C. Private
D. None
Incorrect. The default access specifier for class members in C++ is private.
Correct! The default access specifier for class members in C++ is private.

3. Which operator is used to access members of a class through a pointer?

A. .
B. ->
C. ::
D. &
Incorrect. The correct operator to access members of a class through a pointer is '->'.
Correct! The correct operator to access members of a class through a pointer is '->'.

4. What is the output of the following code snippet?

int main() {

                  int x = 10;

                  cout << x++ << " " << ++x;

                  return 0;

                }
A. 10 12
B. 10 11
C. 11 11
D. 11 12
Incorrect. The output of the code snippet is '10 12'.
Correct! The output of the code snippet is '10 12'.

5. How do you declare a constant variable in C++?

A. const int x = 10;
B. static int x = 10;
C. final int x = 10;
D. readonly int x = 10;
Incorrect. The correct way to declare a constant variable in C++ is 'const int x = 10;'.
Correct! The correct way to declare a constant variable in C++ is 'const int x = 10;'.

6. What is the purpose of the 'this' pointer in C++?

A. To point to the base class
B. To access the object of a class
C. To reference static members
D. To reference the base class
Incorrect. The 'this' pointer is used to access the object of a class.
Correct! The 'this' pointer is used to access the object of a class.

7. Which of the following is not a valid variable name in C++?

A. _variable
B. variable_1
C. 1_variable
D. variable1
Incorrect. '1_variable' is not a valid variable name in C++.
Correct! '1_variable' is not a valid variable name in C++.

8. How do you create a single-line comment in C++?

A. <!-- Comment -->
B. /* Comment */
C. // Comment
D. # Comment
Incorrect. The correct way to create a single-line comment in C++ is '// Comment'.
Correct! The correct way to create a single-line comment in C++ is '// Comment'.

9. What is the use of the 'new' operator in C++?

A. To allocate memory on the stack
B. To deallocate memory
C. To allocate memory on the heap
D. To create an object
Incorrect. The 'new' operator is used to allocate memory on the heap.
Correct! The 'new' operator is used to allocate memory on the heap.

10. What is the purpose of the 'main' function in C++?

A. To define a class
B. To start program execution
C. To include header files
D. To create objects
Incorrect. The 'main' function is used to start program execution.
Correct! The 'main' function is used to start program execution.
