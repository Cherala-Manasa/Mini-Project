ProgQuiz - CSS Language Beginner Level

Beginner CSS Language Quiz

Test your basic knowledge of CSS.

1. What does CSS stand for?

A. Creative Style Sheets
B. Colorful Style Sheets
C. Computer Style Sheets
D. Cascading Style Sheets
Incorrect. The correct answer is D. CSS stands for Cascading Style Sheets.
Correct! CSS stands for Cascading Style Sheets.

2. How do you select an element with the class name "example" in CSS?

A. #example
B. .example
C. *example
D. example
Incorrect. The correct answer is B. To select an element with the class name "example," use .example.
Correct! Use .example to select elements with the class name "example."

3. Which property is used to change the text color of an element?

A. font-color
B. color
C. text-color
D. background-color
Incorrect. The correct answer is B. The 'color' property is used to change the text color of an element.
Correct! Use the 'color' property to change the text color of an element.

4. How do you add a background color to an element in CSS?

A. background
B. color
C. bg-color
D. background-color
Incorrect. The correct answer is D. Use the 'background-color' property to add a background color to an element.
Correct! The 'background-color' property adds a background color to an element.

5. What is the correct CSS syntax to make all <p> elements bold?

A. p { font-bold: true; }
B. p { text-weight: bold; }
C. p { bold: yes; }
D. p { font-weight: bold; }
Incorrect. The correct answer is D. Use 'font-weight: bold;' to make all <p> elements bold.
Correct! The 'font-weight: bold;' property makes all <p> elements bold.

6. What is the output of the following CSS code?

p {
  color: red;
  font-size: 20px;
}
A. Text color is blue and font size is 20px
B. Text color is red and font size is 16px
C. Compilation error
D. Text color is red and font size is 20px
Incorrect. The correct answer is D. The 'color' is red and 'font-size' is 20px.
Correct! The 'color' property sets the text color to red, and 'font-size' sets the font size to 20px.

7. What is the result of the following CSS code?

h1 {

                  text-align: center;

                  margin: 10px;

                }
A. The text in <h1> is left-aligned with no margin
B. The text in <h1> is right-aligned with a margin of 10px
C. The text in <h1> is centered with a margin of 0px
D. The text in <h1> is centered with a margin of 10px
Incorrect. The correct answer is D. The text in <h1> is centered with a margin of 10px.
Correct! The 'text-align' property centers the text, and the 'margin' property sets a 10px margin.

8. How do you select all <div> elements within a <section> element?

section div {

                  /* CSS properties */

                }
A. div section
B. section + div
C. section div
D. section > div
Incorrect. The correct answer is C. 'section div' selects all <div> elements inside a <section> element.
Correct! 'section div' selects all <div> elements within a <section> element.

9. What does the 'float' property do in CSS?

A. It centers an element horizontally
B. It changes the element's font size
C. It allows an element to float to the left or right of its container
D. It makes an element fixed at a specific position
Incorrect. The correct answer is C. The 'float' property allows an element to float to the left or right of its container.
Correct! The 'float' property makes an element float to the left or right of its container.

10. Which CSS property is used to control the spacing between lines of text?

A. text-spacing
B. font-spacing
C. line-height
D. margin
Incorrect. The correct answer is C. 'line-height' controls the spacing between lines of text.
Correct! The 'line-height' property controls the spacing between lines of text.
