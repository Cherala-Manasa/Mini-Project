C++ Language Intermediate Quiz

Test your intermediate knowledge of C++ programming Language.

1. Which of the following is used to handle exceptions in C++?

A. catch
B. throw
C. try
D. All of the above
Incorrect. The correct answer is 'D. All of the above'. All of these keywords are used in exception handling in C++.
Correct! All of these keywords are used in exception handling in C++.

2. What is the purpose of the 'virtual' keyword in C++?

A. To create static member functions
B. To enable function overloading
C. To ensure runtime polymorphism
D. To define a constant
Incorrect. The correct answer is 'C. To ensure runtime polymorphism'. The 'virtual' keyword is used to achieve runtime polymorphism in C++.
Correct! The 'virtual' keyword is used to ensure runtime polymorphism in C++.

3. Which of the following containers is not part of the C++ Standard Library?

A. vector
B. list
C. array
D. map
Incorrect. The correct answer is 'C. array'. The Standard Library container classes are vector, list, and map, but array is not part of it.
Correct! The 'array' container is not part of the C++ Standard Library. The correct containers are vector, list, and map.

4. What is the purpose of the 'friend' keyword in C++?

A. To allow access to private members of a class
B. To create a private class
C. To declare a global function
D. To protect members of a class
Incorrect. The correct answer is 'A. To allow access to private members of a class'. The 'friend' keyword grants access to private and protected members of a class.
Correct! The 'friend' keyword is used to allow access to private and protected members of a class.

5. What does the 'mutable' keyword allow you to do in C++?

A. Modify const member variables
B. Change the access specifier
C. Create a new variable
D. Define a static member
Incorrect. The correct answer is 'A. Modify const member variables'. The 'mutable' keyword allows modification of member variables even if they are in a 'const' member function.
Correct! The 'mutable' keyword allows modification of const member variables.

6. What is the output of the following code snippet?

int main() {

                  int a = 5, b = 10;

                  cout << a << " " << b << endl;

                  swap(a, b);

                  cout << a << " " << b << endl;

                  return 0;

                }
A. 5 10
B. 10 5
C. 5 5
D. 10 10
Incorrect. The correct answer is 'B. 10 5'. The 'swap' function exchanges the values of 'a' and 'b'.
Correct! The 'swap' function exchanges the values of 'a' and 'b', so the output is '10 5'.

7. What is the output of the following code snippet?

int main() {

                  int x = 1;

                  int y = x++ + ++x;

                  cout << y << endl;

                  return 0;

                }
A. 3
B. 4
C. 5
D. 6
Incorrect. The correct answer is 'B. 4'. The value of 'x' is incremented twice, resulting in 'y' being 4.
Correct! The output is '4'. The expression involves both post-increment and pre-increment operators.

8. What is the output of the following code snippet?

int main() {

                  int arr[3] = {1, 2, 3};

                  for (int i = 0; i < 3; i++)

                    cout << arr[i] << " ";

                  return 0;

                }
A. 1 2 3
B. 2 3 4
C. 1 1 1
D. 3 2 1
Incorrect. The correct answer is 'A. 1 2 3'. The code prints the elements of the array in order.
Correct! The output is '1 2 3'. The for loop prints each element of the array.

9. What is the output of the following code snippet?

int main() {

                  int a = 3;

                  int b = a * ++a;

                  cout << b << endl;

                  return 0;

                }
A. 12
B. 9
C. 15
D. 6
Incorrect. The correct answer is 'A. 12'. The value of 'a' is incremented before multiplication, resulting in 'a * a' being 12.
Correct! The output is '12'. The pre-increment operator increases 'a' to 4 before multiplication.

10. What is the output of the following code snippet?

int main() {

                  int x = 5;

                  int &y = x;

                  y += 10;

                  cout << x << " " << y << endl;

                  return 0;

                }
A. 15 15
B. 5 15
C. 15 5
D. 5 5
Incorrect. The correct answer is 'A. 15 15'. The reference 'y' points to the same variable 'x', so both show the updated value.
Correct! Both 'x' and 'y' refer to the same variable, so both will display the updated value '15'.
