Intermediate CSS Language Quiz

Test your intermediate knowledge of CSS.

1. What is the default value of the 'position' property in CSS?

A. relative
B. absolute
C. fixed
D. static
Incorrect. The correct answer is D. The default value of the 'position' property is 'static'.
Correct! The default value of the 'position' property is 'static'.

2. How do you apply a CSS transition effect to an element?

A. animate: property duration;
B. effect: property duration;
C. transition-effect: property duration;
D. transition: property duration;
Incorrect. The correct answer is D. The 'transition' property is used to apply a CSS transition effect.
Correct! The 'transition' property is used to apply a CSS transition effect.

3. What does the 'flex-grow' property do in CSS Flexbox?

A. Specifies the item's alignment
B. Specifies the item's maximum size
C. Specifies how much the item will grow relative to the rest
D. Specifies the item's minimum size
Incorrect. The correct answer is C. The 'flex-grow' property specifies how much the item will grow relative to the rest of the flex items.
Correct! The 'flex-grow' property specifies how much the item will grow relative to the rest of the flex items.

4. How do you create a CSS grid layout with 3 columns?

A. grid-columns: 3;
B. grid-template: 3 columns;
C. column-count: 3;
D. grid-template-columns: repeat(3, 1fr);
Incorrect. The correct answer is D. 'grid-template-columns: repeat(3, 1fr);' creates a grid layout with 3 columns.
Correct! 'grid-template-columns: repeat(3, 1fr);' creates a grid layout with 3 columns.

5. What is the purpose of the 'z-index' property in CSS?

A. Specifies the visibility of elements
B. Specifies the stack order of positioned elements
C. Specifies the opacity of elements
D. Specifies the margin of elements
Incorrect. The correct answer is B. The 'z-index' property specifies the stack order of positioned elements.
Correct! The 'z-index' property specifies the stack order of positioned elements.

6. What is the result of the following CSS code?

button {
  background-color: blue;
  color: white;
  border: none;
}
A. Blue text with white background and no border
B. Blue background with white text and no border
C. White background with blue text and no border
D. Blue background with default text color and border
Incorrect. The correct answer is B. The background color is blue, text color is white, and there is no border.
Correct! The background color is blue, text color is white, and there is no border.

7. What does the 'calc()' function do in CSS?

div {

                  width: calc(100% - 20px);

                }
A. Calculates the number of elements
B. Performs mathematical operations on color values
C. Performs calculations to determine CSS property values
D. Calculates the font size based on screen resolution
Incorrect. The correct answer is C. The 'calc()' function performs calculations to determine CSS property values.
Correct! The 'calc()' function performs calculations to determine CSS property values.

8. How can you create a media query for devices with a maximum width of 768px?

@media (max-width: 768px) {

                  /* CSS properties */

                }
A. @media (min-width: 768px)
B. @media (width: 768px)
C. @media (device-width: 768px)
D. @media (max-width: 768px)
Incorrect. The correct answer is D. The media query '@media (max-width: 768px)' targets devices with a maximum width of 768px.
Correct! The media query '@media (max-width: 768px)' targets devices with a maximum width of 768px.

9. What is the use of the 'opacity' property in CSS?

A. It sets the color of an element
B. It sets the transparency level of an element
C. It sets the border width of an element
D. It sets the font style of an element
Incorrect. The correct answer is B. The 'opacity' property sets the transparency level of an element.
Correct! The 'opacity' property sets the transparency level of an element.

10. Which CSS property is used to control the spacing between lines of text?

A. text-spacing
B. font-spacing
C. line-height
D. margin
Incorrect. The correct answer is C. 'line-height' controls the spacing between lines of text.
Correct! The 'line-height' property controls the spacing between lines of text.
