Intermediate JavaScript Quiz

Test your intermediate knowledge of JavaScript.

1. What will the following code output?

console.log([1] == [1]);
A. true
B. false
C. ReferenceError
D. TypeError
Incorrect. The correct answer is B. Two different array instances are not equal, even if their contents are the same.
Correct! The result is false because [1] and [1] are different array instances, and they are not equal.

2. What is the output of this code snippet?

console.log(typeof null);
A. 'object'
B. 'null'
C. 'undefined'
D. 'number'
Incorrect. The correct answer is A. The typeof null returns 'object', which is a historical bug in JavaScript.
Correct! The typeof null returns 'object', which is a historical bug in JavaScript.

3. What will this function return?

function foo(a, b) {
  return arguments.length;
}
console.log(foo(1, 2, 3));
A. 2
B. 3
C. 4
D. undefined
Incorrect. The correct answer is B. The arguments object contains all the arguments passed to the function, so its length is 3.
Correct! The arguments object contains all the arguments passed to the function, so its length is 3.

4. What will the following code output?

console.log(0.1 + 0.2 === 0.3);
A. true
B. false
C. NaN
D. undefined
Incorrect. The correct answer is B. Due to floating-point precision issues, 0.1 + 0.2 does not exactly equal 0.3.
Correct! Due to floating-point precision issues, 0.1 + 0.2 does not exactly equal 0.3.

5. How do you define a class in JavaScript?

A. class MyClass {}
B. function MyClass() {}
C. var MyClass = class {}
D. Both A and C
Incorrect. The correct answer is D. Classes can be defined using the class keyword (A) or as a class expression (C).
Correct! Classes can be defined using the class keyword (A) or as a class expression (C).

6. What will be the result of the following code?

const obj = { a: 1 };
const copy = { ...obj };
copy.a = 2;
console.log(obj.a);
A. 1
B. 2
C. undefined
D. ReferenceError
Incorrect. The correct answer is A. The spread operator creates a shallow copy of the object, so changes to the copy do not affect the original.
Correct! The spread operator creates a shallow copy of the object, so changes to the copy do not affect the original. Therefore, obj.a remains 1.

7. What will this code output?

let x = 1;
setTimeout(() => console.log(x), 1000);
x = 2;
A. 1
B. 2
C. undefined
D. ReferenceError
Incorrect. The correct answer is B. By the time the setTimeout callback runs, x has already been updated to 2.
Correct! By the time the setTimeout callback runs, x has already been updated to 2.

8. What is the output of this code snippet?

const add = (a, b = 1) => a + b;
console.log(add(2));
A. 3
B. 2
A TypeError
undefined
Incorrect. The correct answer is A. The default value of b is used when it's not provided, so the result is 2 + 1 = 3.
Correct! The default value of b is used when it's not provided, so the result is 2 + 1 = 3.

9. What is the result of the following code?

function* gen() {
  yield 1;
  yield 2;
  return 3;
}
const iterator = gen();
console.log(iterator.next());
console.log(iterator.next());
console.log(iterator.next());
A. {value: 1, done: false}, {value: 2, done: false}, {value: 3, done: true}
B. {value: 1, done: false}, {value: 2, done: false}, {value: undefined, done: true}
C. 1, 2, 3
D. {value: 1, done: true}, {value: 2, done: true}, {value: 3, done: true}
Incorrect. The correct answer is A. The generator yields values 1 and 2, then returns 3 when the iteration is complete.
Correct! The generator yields values 1 and 2, then returns 3 when the iteration is complete.
